# Annex E – Standard Requests List

The table below contains the list of requests considered standard. Any other request fulfilment will be classified as “non-standard”.

| Request Name | Scope | Exclusions | Average estimated time required per request (hour) | Included # requests in scope per month |
| :--- | :--- | :--- | :---: | :---: |
| **Edit existing Classes** | FBVN will re-structure/re-configure and/or assist editing existing Classes as defined in Annex D: <br> - Add new Properties <br> - Edit Properties’ information | - Changes that caused error to running workflow(s) and/or configuration in Compliance Kit application <br> - Configure for Properties’ Automatic Value | 2 | 5 |
| **Add new Classes/ Objects** | FBVN will create and/or assist to create new Objects/Classes. Each newly created Class should be included within 15 Properties | - Configure for Properties’ Automatic Value | 2 | 5 |
| **Auto-calculated Properties Value** | FBVN will configure and/or assist to configure the Automatic Value of Property as M-FILES’ standard feature, included: <br> - Simple Automatic Numbering <br> - Customized Automatic Numbering (VbScript) <br> - Simple Concatenating of Properties <br> - Calculated Value (VbScript) | - Changes that caused error to running workflow(s) and/or configuration in Compliance Kit application <br> - Variable’ values which are captured from another objects/ workflow states, and/or requires function that not in standard of M-FILES | 8 | 2 |
| **Edit existing Workflows** | FBVN will re-structure/re-configure and/or assist editing existing Workflows as defined in Annex D, includes: <br> - Workflow’ states, transition <br> - Permission <br> - Notification <br> - Set Properties <br> - Convert to PDF format <br> - Assignment <br> - Post/Pre-Condition | - Changes that caused error to running workflow(s) and/or configuration in Compliance Kit application <br> - Configure Post/Pre-Condition (VbScript), & VbScript in Workflows [Action] | 8 | 2 |
| **Add new Workflows** | FBVN will create and/or assist to create new Workflows. Each newly created Workflow should be included within 12 States | - Changes that caused error to running workflow(s) and/or configuration in Compliance Kit application <br> - Configure VbScript in Workflows [Action] & Post/Pre-Condition (VbScript) that requires more than 03 logical functions/ procedures (calculations, conditionals,…), and/or requires value input/output from/to more than 03 properties/classes | 15 | 1 |
| **Edit existing Access Control Lists (ACLs)** | FBVN will re-structure/re-configure and/or assist editing existing ACL(s) as defined in Annex D | - Changes that caused error to running workflow(s) and/or configuration in Compliance Kit application | 2 | 2 |
| **Add new Access Control Lists (ACLs)** | FBVN will create and/or assist to create new ACLs. Each newly created ACL should be included within 10 Users/Groups | | 2 | 2 |
| **Migrate M-Files system** | FBVN will assist to migrate the M-Files system to new infrastructure; and support to investigate the problem from M-Files (if any) in migration activity. | All errors are from hardware/outside M-Files system | 8 | 1 |
| **M-Files - Salesforce integration maintenance** | FBVN will assist to configure and maintain the integration between M-FILES & Salesforce system. | All errors are from Salesforce system | 4 | 1 |
| **User & Admin training** | FBVN will deliver the training session of how to use M-FILES Standard features and Workflow operation as listed in Annex D | - Onsite training | 1 | 2 |
